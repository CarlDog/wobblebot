# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as
its code of conduct. The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

The summary: be respectful. Disagree about technical decisions
freely; don't make it personal. Treat the issue tracker, PR
discussions, and any other project space as a professional venue.

## Reporting

Conduct concerns can be raised privately by opening a GitHub issue
titled "Conduct: please contact me privately" (without disclosing
details), and the maintainer will reach out via the contact
information on your GitHub profile.

For security vulnerabilities (a separate concern), see
[`SECURITY.md`](SECURITY.md) instead.

## Enforcement

The maintainer follows the Contributor Covenant's enforcement
guidelines (linked above). Consequences scale with the severity and
pattern of the behavior, as described in the canonical text.
