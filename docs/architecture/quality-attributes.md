# Quality Attributes

This system prioritizes:

## 1. Safety
- Strict module separation  
- No uncontrolled money movements  
- Orchestrator validates all actions  

## 2. Determinism
- Bot Core produces identical behavior given identical inputs  
- No randomness unless explicitly injected  

## 3. Extensibility
- Modular ports allow new adapters or modules  
- Easy to add coins, strategies, or exchanges  

## 4. Observability
- Logs every event  
- Metrics exportable  
- Dashboard integration  

## 5. Reliability
- Scheduled routines  
- Controlled retries  
- Minimal external dependencies
